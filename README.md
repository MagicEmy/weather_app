# Weather Application - Module 01

This repository contains the first module of a 3-part series dedicated to building a complete Weather Application.

## 📱 Project Overview
**Weather App (Modules 01-03):** Focusing on UI, API integration, and design.

---

## 🗂️ Weather App Project Structure

### Module 01: Structure and Logic ✅ CURRENT
**Project Name:** `weather_app`
- Basic application structure with bottom navigation (Currently, Today, Weekly).
- Top app bar with search functionality and geolocation button.
- Tab switching logic (tap and swipe).

### Module 02: API and Data Management 🔄
**Project Name:** `medium_weather_app`
- GPS geolocation and OpenMeteo API integration.
- Dynamic city search with suggestions.

### Module 03: Design 📋
**Project Name:** `advanced_weather_app`
- Data visualization (charts/graphs) and professional UI/UX polish.

---

## 🛠️ Technology Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **API:** OpenMeteo (Weather & Geocoding)

## 🎯 Learning Objectives (Module 01)

- Mobile application navigation patterns.
- UI component organization with Compose.
- State management and user input handling.
